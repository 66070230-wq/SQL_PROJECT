Project Overview :
This project demonstrates a full end-to-end SQL analytics workflow using a synthetic but realistic business dataset.
The goal is to analyze:
- Customer behavior
- Sales performance
- Product profitability
- Order trends
- Marketing campaign effectiveness

Learning Objectives :
By completing this project, you will demonstrate your ability to:
- Cleanly write SQL queries with clear comments
- Join multiple related tables
- Use aggregations for business metrics
- Analyze customers, products, and revenue
- Build CTEs to break complex logic
- Write CASE conditions for segmentation
- Use date functions for trend analysis
- Apply subqueries for filtering and comparisons
- Combine datasets using UNION
- Produce insights similar to a business analytics role

SQL Topics Covered :
- SELECT, WHERE, ORDER BY
- Aggregations: SUM, COUNT, AVG, MIN, MAX
- JOINs (INNER, LEFT)
- Handling NULL
- Arithmetic + custom calculations
- Date functions (DATE_TRUNC, EXTRACT, CASE)
- Subqueries (inline + correlated)
- CTEs (WITH) for logic organization
- UNION + UNION ALL
- Basic data manipulation concepts
